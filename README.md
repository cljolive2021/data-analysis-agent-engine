# Data Analysis Engine - Copilot Studio

Motor de análise de dados 100% gratuito no Vercel com Next.js

## 🎯 Funcionalidades

- ✅ Upload de arquivos (CSV, Excel)
- ✅ Análise automática de correlações
- ✅ Insights inteligentes com recomendações
- ✅ Gráficos interativos em tempo real
- ✅ Interface web responsiva
- ✅ 100% gratuito no Vercel

## 🚀 Deploy Vercel (2 Minutos)

### 1. GitHub
```bash
git add .
git commit -m "Deploy ready"
git push origin main
```

### 2. Vercel
1. Acesse https://vercel.com
2. Clique "Add New" → "Project"
3. Selecione seu repositório
4. Root Directory: `./`
5. Framework: Next.js
6. Deploy automático ✅

**Pronto! Seu app está online!**
